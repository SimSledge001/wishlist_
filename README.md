## Required Features

The following **required** functionality is completed:

* [ ] User can add an item to the wishlist. An item should include: Name, Price, URL
* [ ] User can see their list of items based on previously inputted items
* [ ] New items should be added to a list. If items go past the screen's edge, the list should be scrollable
